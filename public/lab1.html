17. Создайте страницу, содержащую таблицу (элемент <table>). При нажатии на значение заголовка столбца таблицы, строки таблицы должны сортироваться по этому столбцу. При нажатии на любую строку таблицы (кроме строки заголовка) должен выводиться вопрос «удалить строку таблицы?» и при нажатии кнопки Ok, строка таблицы – удаляться.

Для примера будет использована таблица с примерами компьютерных комплектующих и ценами на них.
Сортировка строк происходит в лексикографическом порядке.
